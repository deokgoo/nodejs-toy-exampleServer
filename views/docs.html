Movie API (create deok9)

API Docs(Page)

  : https://movie-nodejs.herokuapp.com

User Register(Page)

  : https://movie-nodejs.herokuapp.com/register

OR

  : curl --location --request POST 'https://movie-nodejs.herokuapp.com/auth/create' \

    --header 'Content-Type: application/x-www-form-urlencoded' \

    --data-urlencode 'id={id}' \

    --data-urlencode 'pw={pw}' \

    --data-urlencode 'name={name}' \

    --data-urlencode 'email={email}'

User Login

  : curl -u {id}:{password} --request POST https://movie-nodejs.herokuapp.com/auth/login

    - return bearer token


SearchMovie (need Bearer token)

   : curl https://movie-nodejs.herokuapp.com/api/search/{title} -H 'Authorization: Bearer token'


Ranking (need Bearer token)

   : curl https://movie-nodejs.herokuapp.com/api/rank -H 'Authorization: Bearer token'
